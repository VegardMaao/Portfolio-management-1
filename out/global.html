createNavigation(container, array)

Creates a series of li elements with links to otherpages. Used for both navigation in header and footer
Parameters:
Name Type Description
container object DOM element
array array an array containing; links, target and a title

// Define what DOM-element the li elements should be produced within, then makes said li elements
